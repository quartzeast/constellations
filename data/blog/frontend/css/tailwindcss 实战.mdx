## 介绍

Tailwind CSS 是一个实用的 CSS 框架，它提供了一组预定义的样式类，可以帮助你快速构建现代的 Web 界面。Tailwind CSS 的核心理念是**不要重复造轮子**，它提供了一组类名，可以让你快速构建出现代的 Web 界面。
